Here are some miscellaneous projects that I've started (and ocassionally, finished!). Also, see my written and musical work.


finished work

  • suan-dl (November 2016) is a small python tool for downloading mixtapes from suan.fm.
  • pokemon aesthetique (December 2016) is a little html/javascript tool for making Super Aesthetic pokemon teams. I wrote this because everyone else was doing it by hand, and I was too lazy to do that.


unfinished work

  • orangeblanket (August 2015 - present): An experiment in creating a 2D adventure game engine. Meant to emulate text-heavy graphical adventure games such as the Famicom Detective Club and Ace Attorney games. Largely abandoned due to lack of interest.
  • Sclera (August 2016? - present): A 2D game engine project that builds on a lot of past similar projects that I eventually abandoned. Developed for a college project, largely as an attempt to implement proper software design practices for once in my life. Abandoned once I was done with the course.
